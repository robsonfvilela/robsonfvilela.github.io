Incorporar vídeo#

Para que o Hugo exiba vídeo do Youtube embutido em um post, é necessário habilitar a opção “unsafe” no arquivo de configuração hugo.toml, como mostrado aqui.

[markup]
    [markup.goldmark.renderer]
      hardWraps = false
#      unsafe = false
      unsafe = true
      xhtml = false
